About

The purpose of this glossary is to create a shared understanding in the travel analysis industry of the terms associated with dynamic passenger assignment (especially for transit).

How

The glossary was created with the following process:

  1. First draft created by the SHRP2-C10 Implementation Project led by the Metropolitan Transportation Commission (MTC)
  2. Draft sent out for comment and review to invited reviewers
    3. MTC team revised glossary based on comments
  3. Public reivew, including a public announcement via TMIP, and ResearchGate
  4. Posting of reviewed content on this site.
  5. Continued refinement, corrections, and additions as suggested and debated in this site's "issues"
